# LLM Codegen Workflow

An interactive presentation documenting Harper Reed's LLM codegen workflow.

**Live Demo:** Deploy to GitHub Pages and share!

## What's Included

- `index.html` - Complete self-contained presentation (deploy directly to GitHub Pages)
- `llm-codegen-workflow.jsx` - React component version (for integration into existing React apps)

## Quick Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html` to the root
3. Go to **Settings → Pages**
4. Set source to "Deploy from a branch" → select `main` branch
5. Your site will be live at `https://[username].github.io/[repo-name]/`

## Features

- **15 slides** covering the complete workflow
- **Keyboard navigation** (← → or Space)
- **Copy-to-clipboard** for all prompts
- **Responsive design** (works on mobile)
- **Dark theme** with amber accents
- **Table of contents** navigation (desktop)

## Content Covered

1. **Greenfield Path** - Full planning workflow
   - Idea honing (interview method)
   - Blueprint generation (TDD & non-TDD prompts)
   - Execution loop (Claude & Aider)

2. **Brownfield Path** - Incremental improvements
   - Context gathering (repomix)
   - Iteration workflow

3. **Prompt Library**
   - Code review
   - GitHub issue generation  
   - Missing tests

4. **Tips & Gotchas**
   - Knowledge cutoff handling
   - Embedding lessons
   - Managing complexity
